> *"Just empty code."*

We build open-source tools, revive dead projects, and mess around with experimental software. No corporate sugarcoating, no buzzwords. We just write the code, fork what needs fixing, and port what needs to run somewhere else. Based in France.

🌐 **Website:** [hexadecinull.github.io](https://hexadecinull.github.io/)

---

## 🛠 What We Do (No Bullshit)

We build whatever we feel like building. From bringing abandoned Android clients back from the dead to writing experimental web proxies, emulators, and font decompilers. Our tech stack is whatever gets the job done—C, C++, Rust, JavaScript, Kotlin, Python, PHP, or Svelte.

Expect to see a lot of *"W.I.P"* and *"Experimental"* tags. If it breaks, submit an issue or fix it yourself.

## 📦 Notable Projects

### 🧟 The Revivals & Ports

* **[VHBB-Android-Revived](https://github.com/Hexadecinull/VHBB-Android-Revived)** — You know that dead VitaDB client for Android? We dug it up, updated it, and made it fully functional again.
* **[ruffle4consoles](https://github.com/Hexadecinull/ruffle4consoles)** — Ruffle (the Flash emulator) ported directly to consoles.
* **[UWP.js](https://github.com/Hexadecinull/UWP.js)** — An online, open-source Unity Web Player emulator. Heavily W.I.P.
* **[qBittorrent-android](https://github.com/Hexadecinull/qBittorrent-android)** — A fork of qBittorrent ported for Android.

### 🧰 Tools & Utilities

* **[Fonty](https://github.com/Hexadecinull/Fonty)** & **[FontyDE](https://github.com/Hexadecinull/FontyDE)** — An open-source font converter for desktops and web browsers, plus a decompilation variant built in C++ to turn fonts back into raw source files. Experimental.
* **[wTorrent](https://github.com/Hexadecinull/wTorrent)** — An open-source, web-based torrent client.
* **[MediaFolderPro](https://www.google.com/search?q=https://github.com/Hexadecinull/MediaFolderPro)** — A straightforward bulk downloader for MediaFire, written in Python.

### 🌐 Web & Streaming

* **[Streamsuite](https://github.com/Hexadecinull/Streamsuite)** — A free, ad-less, and open-source streaming client for the web. Because modern streaming ads are garbage.
* **[WebSquared](https://github.com/Hexadecinull/WebSquared)** — A free and open-source web proxy for browsers built in Svelte.
* **[MindMyMap](https://github.com/Hexadecinull/MindMyMap)** — An open-source GitMind alternative, picking up where La Digitale left off.
